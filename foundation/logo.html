Variants

Color

This full color logo is the preferred usage for all appearances of the logo when creating collateral. Full Color can be used on white or dark colored backgrounds.

Grayscale

In instances where printing in color is not an option, consider using the logo in grayscale form.

Color Inversed

The Color Inversed Logo should be used on colored backgrounds or photographic elements to increase legibility.

Gradient

This Gradient color logo is an option for cross branding to add dynamic feeling. It can be used on white or dark colored backgrounds.

Pride

This Pride logo can be used for Pride months. It can be used on white or dark colored backgrounds.

Neumorphism

This is an experimental logo for a soft UI

Logo Specifications

This logo is designed with the Golden Ratio in mind from the pill dimensions to the text sizes.
